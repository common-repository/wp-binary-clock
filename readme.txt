== Description ==
Binary clock - is a binary clock.
No one, or very few people can read this watch. Therefore it is rather pointless, but a must for nerds, geeks and Tekkis.
Feel free to use it.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Shortcode [tpb_binary_clock] to display the clock.
